> The mod contains a complete set of large grid letters, numbers and symbols (equivalent to the vanilla set) of emissive aurebesh characters in a style similar to neon lights. They are ideal for decorating buildings and interiors in a Star Wars style with.

[![](https://raw.githubusercontent.com/enenra/gfa/master/assets/mod_banner.png)](https://steamcommunity.com/sharedfiles/filedetails/?id=2978742816)

![img](https://raw.githubusercontent.com/enenra/gfa/master/assets/header_features.png)

* Letters A-Z
* Numbers 0-9
* Symbols for And, Apostrophe, Colon, Dot, Exclammation Mark, Hyphen, Question Mark, Underscore.

![](https://raw.githubusercontent.com/enenra/gfa/master/assets/header_notes.png)

* There will be no small grid versions of these blocks.
* Obviously inspired by the excellent [Emissive Letters & Symbols mod by Rayanth](https://steamcommunity.com/sharedfiles/filedetails/?id=3347016415). If you want these for latin letters, check it out!
